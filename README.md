# StudioFlow - Premium Manager v6

تطبيق إدارة الشقق والحجوزات اليومية المتقدم مع نظام فواتير وإدارة المستأجرين.

## الميزات الرئيسية

### 1. نظام تسجيل الدخول والمستخدمين
- تسجيل دخول آمن للمستخدمين
- إنشاء حسابات جديدة
- بيانات تجريبية: `admin / 123456`

### 2. إدارة الوحدات العقارية
- إضافة وحذف وحدات عقارية
- عرض مرئي جميل لكل وحدة
- متابعة الحجوزات والمصروفات لكل وحدة

### 3. نظام الحجوزات
- إضافة حجوزات جديدة مع أسماء الضيوف
- تحديد فترات الحجز (تاريخ البداية والنهاية)
- حساب المبالغ تلقائياً
- تقويم تفاعلي يعرض الأيام المحجوزة

### 4. إدارة المستأجرين (جديد)
- حفظ بيانات المستأجرين الكاملة
- الاسم والهاتف والبريد الإلكتروني ورقم الهوية
- عرض سهل الوصول لكل المستأجرين

### 5. نظام الفواتير (جديد)
- إنشاء فواتير تلقائية من الحجوزات
- تتبع حالة الفاتورة (مدفوعة/معلقة)
- تأكيد الدفعات مباشرة من التطبيق
- عرض مبالغ الفواتير المدفوعة والمعلقة

### 6. إدارة المصروفات
- تسجيل جميع المصروفات
- تصنيف المصروفات حسب النوع
- حساب صافي الربح تلقائياً

### 7. التقارير والإحصائيات (جديد)
- عرض إجمالي الإيرادات
- إجمالي المصروفات
- حالة الفواتير (مدفوعة/معلقة)
- حساب صافي الربح

### 8. الإشعارات (جديد)
- تنبيهات عند إضافة/تحديث البيانات
- رسائل النجاح والخطأ
- إشعارات الدفعات والتحديثات

## كيفية الاستخدام

### تسجيل الدخول
1. افتح التطبيق
2. استخدم البيانات التجريبية:
   - **المستخدم**: admin
   - **كلمة المرور**: 123456

### إضافة وحدة عقارية
1. اضغط على زر "وحدة جديدة" في الهيدر
2. أدخل اسم الوحدة/العقار
3. اضغط "تأكيد الحفظ"

### إضافة حجز
1. اضغط على الوحدة لفتح تفاصيلها
2. اضغط على "حجز جديد"
3. أدخل:
   - اسم الضيف/المستأجر
   - تاريخ البداية
   - تاريخ النهاية
   - المبلغ الإجمالي
4. اضغط "تأكيد الحفظ"

### إضافة مستأجر
1. افتح تفاصيل الوحدة
2. اضغط على "مستأجر جديد"
3. أدخل بيانات المستأجر:
   - الاسم الكامل
   - رقم الهاتف
   - البريد الإلكتروني
   - رقم الهوية

### إنشاء فاتورة
1. في صفحة تفاصيل الوحدة
2. اذهب إلى تبويب "الفواتير"
3. سيتم عرض الفواتير المنشأة
4. يمكنك تأكيد الدفع مباشرة

### عرض التقارير
- اضغط على زر "التقارير" في الهيدر
- سيظهر ملخص مفصل:
  - إجمالي الإيرادات
  - إجمالي المصروفات
  - الفواتير المدفوعة والمعلقة
  - صافي الربح

## التخزين والبيانات

- جميع البيانات تُحفظ محلياً في `localStorage`
- لا تحتاج إلى اتصال بالإنترنت
- البيانات تبقى آمنة في المتصفح

## التصميم والواجهة

- واجهة حديثة بألوان احترافية
- تصميم مستجيب (Responsive)
- رسوم متحركة سلسة
- دعم العربية الكامل (RTL)

## المتطلبات التقنية

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- JavaScript مفعّل
- لا توجد متطلبات إضافية
