# دليل الاختبار - StudioFlow v6

## اختبار الميزات الأساسية

### 1. تسجيل الدخول

#### الحالة 1: تسجيل دخول صحيح ✅
```
المستخدم: admin
كلمة المرور: 123456
النتيجة المتوقعة: دخول ناجح + إشعار ترحيب
```

#### الحالة 2: بيانات خاطئة ❌
```
المستخدم: admin
كلمة المرور: wrongpassword
النتيجة المتوقعة: رسالة خطأ
```

#### الحالة 3: إنشاء حساب جديد ✅
```
اضغط: إنشاء حساب جديد
أدخل: testuser
كلمة المرور: test123
الاسم: مستخدم اختبار
النتيجة: حساب جديد ناجح
```

---

## 2. إدارة الوحدات

### إضافة وحدة ✅
```
1. اضغط "وحدة جديدة"
2. أدخل: فيلا الياسمين
3. اضغط "تأكيد الحفظ"
4. ستظهر الوحدة برسم جميل
5. عدد الوحدات يتحدث في الهيدر
```

### حذف وحدة ✅
```
1. اضغط على الوحدة
2. مرّر الماوس على الدائرة
3. اضغط على أيقونة السلة الحمراء
4. ستختفي الوحدة
```

---

## 3. نظام الحجوزات

### إضافة حجز ✅
```
1. افتح الوحدة
2. اضغط "حجز جديد"
3. ملأ البيانات:
   - الضيف: محمد أحمد
   - البداية: 2024-01-01
   - النهاية: 2024-01-05
   - المبلغ: 2500
4. اضغط "تأكيد"
5. سيظهر الحجز في الجدول
```

### التقويم ✅
```
1. التحقق: الأيام المحجوزة ملونة
2. يجب أن تعكس فترة الحجز
3. الشهر والسنة صحيحة
```

### حذف حجز ✅
```
1. في تبويب "الحجوزات"
2. مرّر على الصف
3. اضغط أيقونة السلة
4. سيختفي الحجز
```

---

## 4. إدارة المستأجرين

### إضافة مستأجر ✅
```
1. من لوحة الوحدة
2. اضغط "مستأجر جديد"
3. ملأ البيانات الكاملة
4. اضغط "تأكيد"
5. سينضم للقائمة
```

### عرض المستأجرين ✅
```
1. اضغط تبويب "المستأجرين"
2. شاهد البطاقات الجميلة
3. كل بطاقة تحتوي على البيانات الكاملة
```

### حذف مستأجر ✅
```
1. في تبويب "المستأجرين"
2. اضغط أيقونة السلة
3. سيتم حذفه مباشرة
```

---

## 5. نظام الفواتير

### إنشاء فاتورة ✅
```
البرنامج يربط الفاتورة تلقائياً مع الحجوزات
يمكنك تحويل الحجز إلى فاتورة معلقة
```

### تأكيد الدفع ✅
```
1. اذهب لتبويب "الفواتير"
2. الفواتير المعلقة باللون الأصفر
3. اضغط "تأكيد الدفع"
4. ستتحول للون الأخضر "مدفوع"
```

### الفواتير المدفوعة ✅
```
1. تظهر بلون أخضر
2. تحتوي على تاريخ الدفع
3. لا يمكن تعديلها
```

---

## 6. المصروفات

### إضافة مصروف ✅
```
1. اضغط "إضافة مصروف"
2. أدخل:
   - البيان: صيانة
   - المبلغ: 500
3. اضغط "تأكيد"
4. سيظهر في القائمة
```

### حذف مصروف ✅
```
1. في تبويب "المصروفات"
2. مرّر الماوس
3. اضغط السلة
4. سيختفي
```

---

## 7. التقارير والإحصائيات

### فتح التقارير ✅
```
1. اضغط "التقارير" في الهيدر
2. ستظهر لوحة جميلة
3. تحتوي على:
   - إجمالي الإيرادات (أزرق)
   - المصروفات (أحمر)
   - الفواتير المدفوعة (أخضر)
   - الفواتير المعلقة (أصفر)
   - صافي الربح (رمادي)
```

### الحسابات ✅
```
صافي الربح = الإيرادات - المصروفات
يجب أن يتحدث تلقائياً عند:
- إضافة حجز
- إضافة مصروف
- تأكيد فاتورة
```

---

## 8. الإشعارات

### إشعارات النجاح ✅
```
تظهر عند:
- إضافة وحدة
- إضافة حجز
- إضافة مستأجر
- تأكيد دفع
- أي عملية ناجحة

النص: "تم [الإجراء] بنجاح"
اللون: أخضر
المدة: 3 ثوان
```

### إشعارات الخطأ ✅
```
تظهر عند:
- بيانات غير صحيحة
- حجز غير موجود
- أي عملية فاشلة

اللون: أحمر
المدة: 3 ثوان
```

```
اختبر:

3. افتحه مرة أخرى
```

```
3. شاشة تسجيل الدخول تعود
4. البيانات محفوظة (للدخول لاحقاً)
```
