# دليل البدء السريع - StudioFlow v6

## الخطوات الأولى (5 دقائق)

### 1️⃣ تسجيل الدخول
```
👤 المستخدم: admin
🔐 كلمة المرور: 123456
```
الصق هذه البيانات في شاشة تسجيل الدخول واضغط "دخول"

### 2️⃣ إضافة أول وحدة
1. اضغط "وحدة جديدة" في الزاوية العلوية اليمين
2. اكتب اسم الوحدة (مثلاً: "فيلا الياسمين")
3. اضغط "تأكيد الحفظ"
4. ستظهر الوحدة كـ دائرة جميلة في المنتصف

### 3️⃣ إضافة حجز أول
1. اضغط على الوحدة التي أضفتها
2. ستفتح لوحة التفاصيل
3. اضغط "حجز جديد"
4. ملأ البيانات:
   - **الضيف**: أحمد محمد
   - **البداية**: 2024-01-01
   - **النهاية**: 2024-01-05
   - **المبلغ**: 2500
5. اضغط "تأكيد الحفظ"

### 4️⃣ إضافة مستأجر
1. من نفس لوحة الوحدة
2. اضغط "مستأجر جديد"
3. أدخل البيانات:
   - **الاسم**: أحمد محمد
   - **الهاتف**: 0501234567
   - **البريد**: ahmed@email.com
   - **الهوية**: 1234567890
4. اضغط "تأكيد الحفظ"
5. اذهب لتبويب "المستأجرين" لرؤيته

## 5️⃣ السيناريو الكامل (خطوة بخطوة)

### المرحلة الأولى: الإعداد
```
✅ قم بـ تسجيل الدخول
✅ أضف 2-3 وحدات عقارية
✅ أضف حجوزات لكل وحدة
```

### المرحلة الثانية: إدارة المستأجرين
```
✅ أضف مستأجرين لكل وحدة
✅ احفظ بيانات الاتصال الخاصة بهم
```

### المرحلة الثالثة: الفواتير والدفع
```
✅ اذهب لتبويب "الفواتير"
✅ ستجد الفواتير المرتبطة بالحجوزات
✅ أكد الدفع عند استلام المبلغ
```

### المرحلة الرابعة: المصروفات
```
✅ اضغط "إضافة مصروف"
✅ أضف مصروفات مثل: الصيانة، الكهرباء، التنظيف
✅ شاهد صافي الربح يتحدث تلقائياً
```

### المرحلة الخامسة: التقارير
```
✅ اضغط "التقارير" في الهيدر
✅ شاهد ملخص شامل لأرباحك
✅ تابع الفواتير المدفوعة والمعلقة
```

## 🎨 نصائح للاستخدام الأفضل

### للشقق
```
أسماء مقترحة:
- شقة غرفة واحدة - دور أول
- شقة عائلية - دور ثالث
- فيلا مستقلة - حي القرم
- غرفة مفروشة - دور أرضي
```

### للفترات
```
أنواع الحجوزات:
- حجوزات يومية (1-2 أيام)
- حجوزات أسبوعية (7 أيام)
- حجوزات شهرية (30 يوم)
```

### للأسعار
```
عوامل التسعير:
- موسم سياحي = سعر أعلى
- فترة طويلة = خصم
- حجز مسبق = خصم إضافي
```

## ❓ أسئلة شائعة

### س: هل بيانتي آمنة؟
**ج**: نعم، جميع البيانات تُحفظ محلياً في جهازك فقط، لا تُرسل أي بيانات للإنترنت.

### س: هل أستطيع حذف بيانات؟
**ج**: نعم، اضغط على أيقونة السلة بجانب أي عنصر.

### س: كيف أنشئ حساب جديد؟
**ج**: اضغط "إنشاء حساب جديد" من شاشة تسجيل الدخول.

### س: هل أحتاج إنترنت؟
**ج**: لا، التطبيق يعمل بدون إنترنت 100%.

### س: ماذا لو أغلقت المتصفح؟
**ج**: جميع البيانات تبقى محفوظة، ستراها عند فتح التطبيق مرة أخرى.

### س: كيف أصدر تقارير؟
**ج**: اضغط "التقارير" - ستجد ملخص شامل وسهل الطباعة.

## 🚀 نصائح متقدمة

### 1. إدارة عدة مستخدمين
```
- أنشئ حساب لك ولشريكك
```

```
- أضف كل مصروف يومياً
- تفقد التقارير أسبوعياً
- احسب الربح الحقيقي
```

### 3. تنظيم الوحدات
```
- استخدم أسماء واضحة
- صنف حسب الموقع
- رقم الوحدات بشكل منطقي
```
